2Menu-bar icon · NSStatusBar template icon

The menu-bar icon is a monochrome template image: black in a light menu bar, white in a dark one, auto-inverted by the system. At 18pt a double ring gets muddy, so it collapses to one double-ring outline + center dot — keeping the sitting/water memory while staying legible small. On alert it flips to a filled glyph + red badge. Clicking opens a popover with the same rings + split action keys as the main window, minus the Reminders settings. The enlarged views only show the strokes; it actually renders at 18pt.

1Stand UP! — macOS main window · interaction states · Dock icon

Main window (page): two rings + a settings block. Frosted-glass vibrancy is simulated with a blurred desktop wallpaper showing through; accent #63EBE9. On alert the ring turns red, the center icon breathes, and Start / Skip pills appear right inside the matching ring. In the Dock it collapses to the rings-only icon (outer = sitting, inner = water, center = standing figure) reflecting live progress, with no settings. The Tweaks panel on the right switches accent color and ring width globally.
